# 🚀 Modern Portfolio Design - Coddiction

A clean, minimal, and modern personal portfolio website built with semantic HTML and custom CSS. This project is currently in the **Development Phase**.

## 🛠️ Tech Stack
- **HTML5**: Structured with semantic elements for better SEO and accessibility.
- **CSS3**: Custom styling using Flexbox and CSS Grid.
- **Remix Icon**: Integrated for sleek, vector-based iconography (`ri-xrp-line`).
- **Google Fonts**: Utilizing 'Inter' and 'Fira Code' for a professional developer aesthetic.

### Phase 1: HTML Structure (Assignment 1) ✅
- [x] Set up document metadata and responsive viewport.
- [x] Created a semantic `<nav>` with a custom logo (`Coddiction`).

### Phase 2: CSS Styling & Layout (Assignment 2) ✅
- [x] **Global Reset**: Implemented `box-sizing: border-box` to ensure consistent sizing.
- [x] **Typography**: Integrated Google Fonts with a fallback stack.
- [x] **Navigation**: Used **Flexbox** for a space-between layout and hover transitions.
- [x] **Grid System**: Used **CSS Grid** with `repeat(auto-fit, minmax(250px, 1fr))` to create a responsive project gallery.
- [x] **Interactivity**: Added smooth `translateY` hover effects and box-shadows to project cards.
- [x] **Minimalist Theme**: Applied a soft background (`#fafafa`) and dark accents (`#111`) for a modern look.
